This is Spoofy.

All of Spotify's expansive library with none of the functionality. It is responsive though.

Try it out

Spotify API

Spoofy leverages the Spotify Web API to access its growing library of tracks, albums, playlists, and artist in a platform-agnostic way.

The Frontend

Spoofy is a a single-page web application created using Angular 8. It makes use of routing and templates to dynamically render pages for each and every piece of content in the Spotify library. In addition, Firebase hosts the application.

The Backend

Spoofy's backend is written using Express JS and hosted on Heroku. This is where all the magic of Spoofy begins. The only way to gain access to Spotify content through Spoofy is to give up your Spotify account credentials. Sort of. You have to log into Spotify at their unique endpoint, then in return, Spoofy's backend gets access tokens to request content from Spotify. That's how it works in a nutshell. The details are a bit more involved so if you're curious, there is more on that here.
